# Dimensionality_Reduction Project

## Overview
This project applies PCA for dimensionality reduction and classification methods for face recognition using the AT&T face database.

## Requirements
```bash
pip install -r requirements.txt
```

## How to Run?
- Run the script:
    ```bash
    python3 main.py
    ```
